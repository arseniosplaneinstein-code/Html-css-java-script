Principais Formatações

Negrito / Destaque

A tag <B> é uma tag ñ Semântica.

exemplo Eu sou muito curuiso as vezes

A tag <STRONG> é uma tag semântica

exemplo Eu estou estudando por voltas das 11:48

Itálico / Ênfase

A tag <i> é uma tag ñ semântica

exemplo Calmo por dentro e tempesdade por fora

A tag <em> é uma tag semântica

sou um tipo de cara que gosta muito de observar as pessoas

Textos Marcados

A tag <mark> é uma tag semantica

xemplo :no dia 15 de outubro de 2007 foi o meu nascimento

exemplo: Gosto muito de estudar programção em casa

Tamanho do texto

A tag <big> serve para colocar uma texto grande.

OBS: Esta tag já ñ pode ser usada porque ela esta obseleta / fora de uso

Exemplo: Eu gosto de jogos de carro,missão e luta

A tag <small> serve para colocar o texto muito mais pequeno

Exemplo: Eu gosto de perfumes suaves e simples como a INTESE MAN

Texto Deletado

A tag <del> serve para riscar um determinado texto numa frase

Exemplo: Eu sou feio

Texto inseridos

A tag <ins> serve para sublinhar um texto numa frase(Semântica)

Exemplo: Vista-se bm, ñ importanta a ocasião

Ou também a tag <u> serve para sublinhar um texto numa frase(Ñ Semântica)

Exemplo: Gosto muito da marca John Foster

Textos Sobrescrito

A tag <sup> serve para elevar um elemento numa frase

Exemplo: 2x2

Textos Sobscrito

A tag <sub> serve para colocar um valor a baixo como por

Exemplo:H2O
